# CLI Reference

`testmcpy` is a single command-line entry point for everything in the toolkit: configuring MCP servers and LLM providers, discovering tools, running YAML test suites, chatting interactively, generating tests, benchmarking models, and gating CI. Every command supports `--help` for the authoritative option list.

```bash
testmcpy [GLOBAL_OPTIONS] COMMAND [COMMAND_OPTIONS]
```

## Global Options

| Option | Description |
|---|---|
| `--version` | Show the installed testmcpy version and exit |
| `--help` | Show help. Also works per command: `testmcpy run --help` |

## Quick Examples

```bash
# Run a test suite against your default MCP profile
testmcpy run tests/

# Run against a specific profile with a CI gate
testmcpy run tests/ --profile prod --gate --junit-xml results.xml

# List the tools your MCP server exposes
testmcpy tools --profile staging --detail

# Chat interactively with an LLM that can call your MCP tools
testmcpy chat --profile prod

# Start the web UI on http://localhost:8000
testmcpy serve

# Generate a starter test suite from a live server
testmcpy generate --mcp-url http://localhost:5008/mcp
```
